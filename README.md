# KKCC的理财本

轻松管理个人收支，让理财变得简单有趣

## 项目简介

这是一个简洁、现代化的个人理财管理网页应用，专为移动端优化设计。帮助用户轻松记录和分析日常收支情况。

## 特性

- 📱 移动端友好的响应式设计
- 🎨 现代化的UI界面
- ⚡ 轻量级，加载速度快
- 🔧 易于扩展和定制

## 技术栈

- HTML5
- CSS3 (响应式设计)
- JavaScript (原生)

## 本地运行

1. 克隆或下载项目
2. 直接在浏览器中打开 `index.html` 文件
3. 或者使用本地服务器（推荐）

## GitHub Pages 部署

1. 将代码推送到 GitHub 仓库
2. 进入仓库设置 (Settings)
3. 找到 Pages 设置
4. 选择 Source 为 "Deploy from a branch"
5. 选择分支为 `main` 或 `master`
6. 点击 Save

部署完成后，你的网站将在 `https://你的用户名.github.io/仓库名` 可访问。
